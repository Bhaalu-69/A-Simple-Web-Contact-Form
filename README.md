## Installation
Please make sure that you have node.js installed in your system before following the next steps.
1. Clone the repository.
2. Run `npm init -y` on your console.
3. Run `npm i` on your console.
4. Change the name of the file `config.json.example` to `config.json` after adding your Mongo DB Connection String inside it at the correct place.(Please make sure that you allow database connections from your server using the MongoDB dashboard)
5. Run `node app.js` on your console.
